# Notebook Note Examples

These Markdown files are examples for `nb` notes. They are intentionally kept
outside the Copier template and `agentsmgr` defaults so they can evolve in this
repository without being copied into every downstream project.

- `team-organization/` contains stable org-chart and topology templates.
- `rolling-handoff.md` is a template for a short checkpoint handoff note.
- `lane-roster.md` is a template for current staffing and lane taxonomy.
- `procedures/` contains procedure-note examples.
